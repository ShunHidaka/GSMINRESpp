Compute Givens rotation parameters for complex values using LAPACK's zlartg routine.

This function computes the parameters of a Givens rotation matrix that eliminates the second entry of a 2-vector. It is a LAPACK-based alternative to zrotg, used in GSMINRES++ to work around known bugs in older versions of OpenBLAS where zrot does not behave correctly.

Parameters
[in,out]fOn input, the first component. On output, replaced with the resulting r.
[in]gOn input, the second component.
[out]cThe cosine of the rotation.
[out]sThe sine of the rotation (complex).
