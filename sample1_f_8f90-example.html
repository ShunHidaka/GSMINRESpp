This example solves a set of shifted linear systems of the form:

\[ (A + \sigma^{(m)} B)x^{(m)} = b, \quad (m=1,\dots,M) \]

using the GSMINRES++ solver.

Matrices A and B are given in Matrix Market format (.mtx) and are loaded as packed U Hermitian matrices. The solver uses BLAS/LAPACK routines internally for matrix-vector multiplication (zhpmv) and linear solves (zpptrf and zpptrs).

A key feature of GSMINRES++ is that the user is free to implement matrix-vector multiplications and linear solves externally. This example shows one such approach using BLAS/LAPACK, but any other representation or computation method can be used as long as the required computational steps are performed in accordance with the expected algorithmic flow.
