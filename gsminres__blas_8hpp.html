Detailed Description

BLAS wrappers for the GSMINRES++ solver.

Author
Shuntaro Hidaka

This header defines lightweight C++ wrapper functions for selected BLAS Level-1 and Level-2 routines such as zaxpy, dznrm2, zdotc, and zhpmv, which are used internally in GSMINRES++. The interfaces are designed for safety and usability using std::vector, and provide explicit control over starting offsets and memory strides for advanced vector operations.
