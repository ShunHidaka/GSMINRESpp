Detailed Description

Header file for the GSMINRES++ solver class.

Author
Shuntaro Hidaka

This header defines the core Solver class used in the GSMINRES++ library. The solver implements the Generalized Shifted MINRES method, capable of simultaneously solving a set of shifted linear systems of the form

\[ (A + \sigma^{(m)} B)x^{(m)} = b, \quad (m = 1, 2, \dots, M) \]

where A is Hermitian, B is Hermitian positive-definite, and the shift parameters \( \sigma^{(m)} \) are complex scalars.
