Detailed Description

LAPACK wrapper functions for the GSMINRES++ solver.

Author
Shuntaro Hidaka

This header provides minimal C++ wrappers for selected LAPACK routines used in sample programs of the GSMINRES++ solver. In particular, it supports operations on packed Hermitian matrices stored in upper-triangular ('U') format.

The routines include Cholesky factorization (zpptrf), linear solve (zpptrs), and robust Givens rotation computation (zlartg) as a workaround for known OpenBLAS issues in zrot.
